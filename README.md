# 🏛️ PPKO Centris

**PPKO Centris** is a powerful, full-stack web application designed for efficient organizational management. Built with a focus on scalability and ease of use, it integrates seamlessly with Supabase for real-time data handling and authentication.

## ✨ Key Features

- **Admin Dashboard**: A centralized interface for managing users, data, and system configurations.
- **Real-time Backend**: Powered by Supabase for instant data synchronization.
- **Modular Architecture**: Clean code structure for easy maintenance and expansion.
- **Secure Authentication**: Robust user login and role-based access control.

## 🛠️ Tech Stack

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP, Supabase
- **Deployment**: Vercel-ready configuration

## 🚀 Getting Started

1. **Clone the repo**: `git clone https://github.com/naylannajwa/ppko-centris.git`
2. **Configure Supabase**: Add your credentials to `config.php`.
3. **Run locally**: Use a local PHP server or deploy directly to Vercel.

## 📁 Directory Structure

- `/admin`: Administrative interface files.
- `/api`: Backend logic and API endpoints.
- `/assets`: Static resources (images, fonts).
- `/config.php`: System configuration settings.
- `/supabase`: Database schemas and migrations.
